# Getting started with iMX7 platform


1) Obtain SBC-iMX7 evaluation board (contact Ilya / Valentin / Dimitry for assistance):
    * CL-SOM-iMX7
    * SB-SOM-iMX7
    * Power Supply
    * USB-to-micro USB cable
    * Ethernet cable (one or two)
    * Startek KD050C-1A LCD or DVI display
2) CL-SOM-iMX7 product [`webpage`](https://www.compulab.com/products/computer-on-modules/cl-som-imx7-freescale-i-mx-7-system-on-module):
3) Find and read:
    * CL-SOM-iMX7 evaluation kit hardware guide
    * Getting started with CL-SOM-iMX7 evaluation kit
4) Download CL-SOM-iMX7 evaluation kit hardware guide, refer to CL-SOM-iMX7 Block Diagram for device architecture
5) Setup and power on device, see console output
6) CL-SOM-iMX7 [`Linux documentation`](https://mediawiki.compulab.com/index.php/CL-SOM-iMX7_Freescale_i.MX7_Linux):
7) Create Live SD card, boot device
8) Connect to wireless network
9) Analog Audio playback

## Required equipment (contact Ilya / Valentin / Dimitry for assistance):
1) SBC-iMX7 with PS and cables, LCD panel or DVI display
2) uSD card + adaptor
3) Card reader
4) USB disk-on-key
5) Analog headphones or speakers
6) WiFi antenna + cables (1 or 2)
