# Post Production Instruction.

## Install the CompuLab image onto the SoM emmc
* [Download the CompuLab Yocto Linux package](https://www.compulab.com/wp-content/uploads/2019/12/cl-som-imx8x_yocto-linux_2019-12-08.zip)
* Extract the “cl-som-imx8x.sdcard.bz2”
* [Create a bootable Sdcard](https://mediawiki.compulab.com/w/index.php?title=CL-SOM-iMX8X:_Yocto_Linux:_Manual_Installation:_SD_card)
* [Installing Yocto images onto the eMMC](https://mediawiki.compulab.com/w/index.php?title=CL-SOM-iMX8X:_Yocto_Linux:_Installing_Yocto_images_onto_the_eMMC)
